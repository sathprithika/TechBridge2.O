# Self_Assistant_AI_Web_App
Ai based self assistant web application

## Frontend

### Installation and usage

```bash
git clone https://github.com/tkdeshan/Self_Assistant_AI_Web_App.git
```
```bash
cd frontend
```
```bash
npm install
```
```bash
npm start
```
